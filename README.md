# Zen Estate Dashboard

Financial dashboard and expense tracking system for real estate property management.

## Overview

This project provides tools for tracking and analyzing property expenses, vendor payments, and financial data across multiple building wings/units.

## Data Privacy

⚠️ **Important**: This repository does not contain actual financial data. Any data files used should be kept private and never committed to version control.

## Setup

1. Clone the repository
2. Add your financial data files (ensure they're listed in `.gitignore`)
3. Process and analyze your data

## File Structure

- `.gitignore` - Prevents sensitive financial data from being committed
- `README.md` - This file

## Contributing

Contributions are welcome! Please ensure no sensitive financial information is included in pull requests.
